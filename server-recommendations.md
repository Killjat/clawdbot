## 快速部署推荐

### 最简单：DigitalOcean
1. 注册账户
2. 创建 Droplet
3. 选择 Ubuntu 22.04
4. 添加 SSH Key 或使用密码
5. 创建后立即可用

1. 新用户有大量优惠
2. 按量付费灵活
3. 国内访问速度快
